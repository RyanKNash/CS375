Q3)
a) The script just has count set to 1, with no code to update it.

let count = 0;
app.get("/count", (req, res) => {
  count = count + 1;
  res.send(`<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Index</title>
  <link	rel="stylesheet" href="/style.css">
</head>
<body>
  <p>This URL has been visited ${count} times.</p>
</body>
</html>`);
});

b) Because the page contains dynamic content that changes on every request.

c) Static sites are simply the pre-written content of the files, exactly as they are stored on the server. 
Dynamic sites generate or modify content on the server in response to a request.
Dynamic sites can display personalized content, track information, read and write to databases, handle login and authentication, etc.
